      • Property Detail

        • Integer debugPort

          This is required, when you want to run the task debugJosm. Set this to the port on which you want to listen for the debug output.

        • boolean isPlugin

          If this is set to true, the *.jar file built for the project is put into the $JOSM_HOME/plugins/ directory for the tasks runJosm or debugJosm

        • String josmCompileVersion

          The version number of JOSM against which the plugin should be compiled.

          Default: The value of the property plugin.compile.version or null if that property is not set.

        • File josmConfigDir

          The directory where the default preferences.xml file is located, which will be used when the directory defined in tmpJosmHome is empty.

          Default value: $projectDir/config/josm

        • final JosmManifest manifest

          The manifest for the JOSM plugin. This is initialized as soon as the project has been evaluated.

        • Closure repositories

          The repositories that are added to the repository list.

          Default value (in this order):

          1. https://josm.openstreetmap.de/download/
          2. https://svn.openstreetmap.org/applications/editors/josm/dist/
          3. https://josm.openstreetmap.de/nexus/content/repositories/releases/
          4. https://josm.openstreetmap.de/nexus/content/repositories/snapshots/

          For details see the source file.

        • File tmpJosmHome

          The directory which should be used as JOSM home directory when executing the runJosm or debugJosm tasks.

          Default value: $buildDir/.josm
